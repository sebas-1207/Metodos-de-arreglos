Método isArray()

El método Array.isArray() en JavaScript se utiliza para determinar si un valor dado es un objeto de tipo "Array". Puedes usar este método para verificar si una variable contiene un arreglo o no. El método devuelve true si el valor es un arreglo y false si no lo es. Aquí hay un ejemplo de cómo se usa: 

                    
                    const miArreglo = [1, 2, 3];
                    const miVariable = 42;
                    console.log(Array.isArray(miArreglo)); // true
                    console.log(Array.isArray(miVariable)); // false

En el ejemplo anterior, Array.isArray(miArreglo) devuelve true porque miArreglo es una instancia de un arreglo, mientras que Array.isArray(miVariable) devuelve false porque miVariable no es un arreglo.
