Método isArray()

Array.isArray() en JavaScript

El método Array.isArray() en JavaScript se utiliza para determinar si un valor dado es un array o no. Es una función que devuelve un valor booleano, true si el valor es un array y false si no lo es. Esto es especialmente útil cuando trabajas con valores dinámicos que pueden ser de diferentes tipos y necesitas asegurarte de que un valor sea un array antes de realizar operaciones específicas de array en él
