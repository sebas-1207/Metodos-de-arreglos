Método join()

El método join() en JavaScript se utiliza principalmente para convertir un array en una cadena de caracteres, donde cada elemento del array se concatena en la cadena resultante, separados por un separador específico o por defecto sin separación. Este método es útil en diversas situaciones, como la generación de cadenas de texto formateadas a partir de datos de un array, la creación de URL a partir de fragmentos, la construcción de consultas SQL dinámicas y muchas otras aplicaciones.

Aquí hay una situaciones comun en las que se utiliza el método join() en JavaScript: 

            
            const frutas = ["manzana", "banana", "cereza"];
            let cadenaFrutas = frutas.join(", ");
            console.log(cadenaFrutas); // "manzana, banana, cereza"
