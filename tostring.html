Método toString()

El método toString() en JavaScript se utiliza para convertir un valor en su representación de cadena de caracteres (string). Este método está disponible en la mayoría de los tipos de datos en JavaScript, incluyendo números, arreglos, objetos y más. Cuando se llama a toString() en un valor, se devuelve una versión de ese valor en formato de cadena de caracteres.

Aquí hay una situaciones comun en las que se utiliza el método toString() en JavaScript: 

                
                const frutas = ["manzana", "banana", "cereza"];
                let cadenaFrutas = frutas.join(", ");
                console.log(cadenaFrutas); // "manzana, banana, cereza"
